# 문법
---

```
// 피연산자 하나
[opcode] [operand]

// 피연산자 둘
[opcode] [operand1], [operand2]
```

<br>

## 예시(C# 디스어셈블리)

```
// 64비트 주소 2874424의 값을 1 증가
inc dword ptr ds:[2874424h]

// EDX 레지스터의 값을 EBP 레지스터의 64비트 주소 3C 위치로 이동
mov dword ptr [ebp-3Ch], edx
```

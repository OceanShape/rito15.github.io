

## Vertex Input

- 입력 조립기 -> 버텍스 쉐이더

```hlsl
struct vi
{
    float4 vertex: POSITION;
    float3 normal: NORMAL;
    float2 uv: TEXCOORD0;
};
```



<br>

## Vertex Output(Vertex to Fragment)

- 버텍스 쉐이더 -> 프래그먼트 쉐이더

```hlsl
struct v2f
{
    float4 vertex: SV_POSITION; // SV : System Value
    float3 normal: NORMAL;
    float2 uv: TEXCOORD0;
};
```

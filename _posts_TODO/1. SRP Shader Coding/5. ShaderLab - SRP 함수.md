
# 공간 변환
---

## TransformObjectToHClip(float3 position)
 - 위치 벡터
 - Object Space -> Clip Space 변환

```hlsl
v2f vert(vi v)
{
    v2f o;
    o.vertex = TransformObjectToHClip(v.vertex.xyz);
    return o;
}
```

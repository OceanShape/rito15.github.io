## **특정 명령어의 도움말 확인하기**

```
git {명령어} -h
```

<br>

## **임의 단축 명령어 만들기**

- 예시 : `git status`를 `git st`로 사용하도록 단축 명령어 등록

```
git config --global alias.st status
```

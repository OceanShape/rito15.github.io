# Init
---

## **로컬 리포지토리로 만들기**

```
git init
```

- `.git` 폴더가 생성된다.

<br>



# Remote
---

## **원격 리포지토리 이름 확인하기**

```
git remote
```

<br>

## **원격 리포지토리 주소 확인하기**

```
git remote -v
```

<br>

## **원격 리포지토리 주소 변경하기**

```
git remote set-url origin [https://~.git]
```

<br>

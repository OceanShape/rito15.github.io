# Clone
---

## **[1] 기본 클론**

```
git clone https://github.com/{사용자명}/{원격 리포지토리명}.git
```

<br>

## **[2] 특정 브랜치만 클론**

```
git clone -b {브랜치명} https://github.com/{사용자명}/{원격 리포지토리명}.git
```

<br>

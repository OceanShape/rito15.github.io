# PAT 사용하여 Push하기
---

- PAT : Personal Access Token

- **예시**
  - 내 Github ID : rito01
  - 내가 가진 토큰(PAT) ID : `abcd1234`
  - 대상 리포지토리 소유자 ID : rito02
  - 대상 리포지토리 이름 : repo01

```
git push https://abcd1234@github.com/rito02/repo01.git
```

# Properties
---

- `Pos X`
  - 비네트 효과가 적용될 중심의 X 좌표
  - 범위 : 0 ~ 1

- `Pos Y`
  - 비네트 효과가 적용될 중심의 Y 좌표
  - 범위 : 0 ~ 1

- `Radius`
  - 비네트 효과 적용 넓이
  - 범위 : 0 ~ 2

- `Smoothness`
  - 비네트 효과 테두리의 부드러운 정도
  - 범위 : 0 ~ 1

- `Width`
  - 비네트 효과 영역의 너비
  - 기본 값 : 1

- `Width`
  - 비네트 효과 영역의 높이
  - 기본 값 : 1
